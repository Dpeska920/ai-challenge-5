Ты — эксперт по пространственному мышлению и физике. Реши следующую задачу, используя топологический анализ объекта.

1. Визуализируй геометрию описанного объекта.

# Мысли по п.2 (Сначала попросите модель дать ответ напрямую)

Простая нейросеть (без включенных размышлений) без дополнительных данных плохо анализирует задачу, и даже крупная модель может ответить неверно

# Мысли по п.3 (Затем в промпте добавьте инструкцию: «решай пошагово»)

Если доработать промпт, давая рекомендацию думать по шагам, то мы включаем простые размышления. Модель не пытается решать задачу целиком, а пошагово обдумывает варианты. Это улучшает результаты, но все еще может не привести к правильному ответу, особенно с мелкими (локальными) нейросетями.

# Мысли по п.4 (Затем попробуйте другой ИИ составить промпт для решения вашей задачи)

Лучше всего для создания промптов изначально использовать крупную модель, которая сможет точно и четко сформулировать задачу. С таким промптом можно добиться правильных ответов даже от мелкой нейросети.

# Мысли по п.5 (Затем создайте внутри ИИ группу экспертов и попросите каждого из них дать решение вашей задачи)

Группе экспертов из крупной нейросети достаточно легко получилось решить задачу. Нейросеть пытается мыслить как каждый из описанных экспертов, что дает разные точки зрения на проблему. Само выделение экспертов уже помогает, т.к. нейросеть вместо абстрактной задачи получает направление, с которого нужно наблюдать. В группе же возможно высказывание сразу нескольких вариантов, которые привносят в контекст разные решения. Это помогает нейросети дать финальный ответ. Но лучше всего это работает, если разделить нейросеть на разные диалоги\контексты, чтобы раздумья были независимыми.

Но, нужно учесть, что маленькой нейросети может все равно не хватить догадливости, чтобы дать правильный ответ, даже с учетом разных специалистов и мнений, она чаще замыкается в своих мыслях, принимает за истину свою ошибочную идею и не может выйти за рамки.
