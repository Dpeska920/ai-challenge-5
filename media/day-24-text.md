# Day 24: Red Pen (Красная Ручка)

## Идея

Telegram-бот для проверки текстов на русском языке. Корректура + факт-чекинг без рерайта — сохраняем авторский стиль.

**На выходе два документа:**
- Чистовик — готовый исправленный текст
- Diff-лог — визуальный changelog с цветовой разметкой изменений

## Фичи

- Принимает текст или файлы (docx, pdf, txt, md)
- Исправляет орфографию и пунктуацию
- Проверяет факты через веб-поиск
- Генерирует Word-документ с визуальным diff (зелёный/красный/оранжевый)
- Персональный словарь (имена, термины которые не трогать)
- Кастомные инструкции пользователя

## Итог

Полностью рабочий проект с Docker Compose. Два микросервиса, интеграция с LLM через OpenAI-совместимый API, self-hosted поиск, генерация документов.
